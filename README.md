# Career Advisor Chatbot

## Project Title
Building Career Advisor Chatbot using Gemini GenAI API

## Project Overview
This is a multi-turn Career Advisor Chatbot built using Google Gemini GenAI API and Streamlit.

The chatbot helps users with:
- Career path suggestions
- Skill roadmaps
- Resume improvement tips
- Interview preparation
- Project ideas
- Career switching guidance

## Domain
Career Advisor Chatbot

## Tech Stack
- Python
- Streamlit
- Google Gemini GenAI API
- python-dotenv

## Features
- Multi-turn conversation
- Chat-style Streamlit interface
- Gemini API integration
- Secure API key handling using environment variables
- Modular API logic
- Structured prompt engineering
- Error handling
- Logging
